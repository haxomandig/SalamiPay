## 2. Purpose

The Branding Expert Agent is the single source of truth for SalamiPay's brand identity. It reviews all design decisions, copy, UI components, and marketing assets against the Official Brand Guidelines V2.0 and flags any deviations.

**Core responsibilities:**

- Enforce brand consistency across all product surfaces

- Audit marketing copy for voice and tone alignment
- Validate logo usage, sizing, and spacing rules
- Ensure naming conventions are followed everywhere
- Provide brand-compliant design recommendations

## 11. Logo System

### Primary Logo
- Structure: Symbol + Wordmark
- Symbol: Interlocking links forming an S shape
- Meaning: Payment link, connection between people, group contribution

### Wordmark
- Font: Inter Bold
- Case: CamelCase — `SalamiPay`

### Logo Variations

| Variation     | Usage                                      |
|---------------|--------------------------------------------|
| Primary       | Green symbol + dark text                   |
| White          | White symbol + white text (dark backgrounds)|
| Icon Only     | Mobile app, favicon, social media          |

### Spacing Rules
- Minimum spacing must equal the height of the letter "S"
- Ensures visual clarity at all sizes

### Size Rules

| Context    | Minimum Width |
|------------|---------------|
| Website    | 120px         |
| Mobile app | 48px          |
| Favicon    | 32px          |

Never scale below readable size. Never crop or distort.

## 12. Brand Color Palette

### Primary Colors

| Color Name     | Usage              | HEX       | RGB              | CMYK               | Meaning                          |
|----------------|--------------------|-----------|------------------|---------------------|----------------------------------|
| Fintech Green  | Primary branding   | `#1F7A5A` | R31 G122 B90     | C75 M20 Y85 K0     | Trust, security, financial credibility |

| Gold           | CTAs, highlighting | `#F4B400` | R244 G180 B0     | C0 M26 Y100 K0     | Call-to-action highlights        |

### Neutral Colors

| Color Name | Usage                | HEX       | RGB              | CMYK               |
|------------|----------------------|-----------|------------------|---------------------|
| Dark       | Primary text         | `#1A1A1A` | R26 G26 B26      | C70 M65 Y65 K75    |
| Grey       | Secondary text       | `#6B7280` | R107 G114 B128   | C49 M39 Y35 K2     |
| Light Grey | Backgrounds, dividers| `#F3F4F6` | R243 G244 B246   | C2 M1 Y0 K0        |
| White      | Backgrounds, text    | `#FFFFFF` | R255 G255 B255   | C0 M0 Y0 K0        |

### Color Usage Rules
- Primary Green (`#1F7A5A`) for brand identity, headers, and nav elements
- Emerald (`#2DBE85`) for interactive elements: buttons, links, progress bars
- Gold (`#F4B400`) used sparingly for CTAs and attention-grabbing highlights only
- Never use more than 3 brand colors in a single composition
- Dark neutrals for text; light neutrals for backgrounds
- Maintain sufficient contrast ratios (WCAG AA minimum)

## 15. UI Design Principles

### Simplicity
Users should be able to create an event in under 30 seconds.

### Transparency
Display clearly: total amount, paid contributors, pending contributors.

### Speed

### Visual Style

| Do Use                    | Don't Use                    |

| Rounded cards             | Heavy textures               |

## 21. Brand Guardrails

### Do

- Keep design minimal and uncluttered, prioritizing ease of use
- Use friendly messaging that emphasizes community and shared moments
- Focus exclusively on event payments and group contribution solutions
- Ensure all communication is clear, direct, and conversational
- Use Inter font family consistently
- Follow the color palette precisely (use HEX values, not approximations)

### Don't

- Use banking terminology or corporate jargon
- Overcomplicate the UI or design flow — every action must be intuitive
- Use too many colors or deviate from the established palette
- Crop or distort the logo or scale it below minimum sizes
- Split the brand name into two words or change its casing
- Use stock photos that feel corporate or impersonal
